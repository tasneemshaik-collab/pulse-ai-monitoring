# PulseAI Monitoring

AI-powered system monitoring dashboard built using the MERN stack.

## Features
- Real-time CPU/RAM monitoring
- AI chatbot assistant
- Email alerts
- JWT Authentication
- Dark/Light mode
- MongoDB persistence

## Tech Stack
React, Node.js, Express.js, MongoDB, JWT, Nodemailer, Groq API

## Run Locally

Frontend:
npm install
npm run dev

Backend:
cd server
npm install
npm run dev
